For this mod you need forge 1.16.4
here is how to download it:
method 1:
1. enter to the website:https://files.minecraftforge.net
2. go to 1.16.4 and click on recommended
3. click on the  Installer
4. wait to the skip button ot show up AND DONT CLICK AT ANY OF THE ADS
5.click on the .jar file
6. go to %appdata% and click on .minecraft
7. create the mods folder in there
8. drag the mod into there
9. open minecraft
method 2:
1. download curseforge
2. create a profile with forge 1.16.4
3. press the three dots button and select vieww profile folder
4. select the mods folder
5. drag the mod to there
